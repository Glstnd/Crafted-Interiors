About the API

The Crafted Interiors API powers our furniture website, providing functionality for browsing furniture catalogs and managing user accounts. It consists of two main microservices:

  • Catalog Service: Handles furniture product listings, categories, and search functionality, enabling users to explore and discover items.
  • Accounts Service: Manages user authentication, registration, and profile management, ensuring secure access to the platform.
